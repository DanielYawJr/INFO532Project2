What is Netsite server software?

Netsite is the first commercial Internet server software that is compatible with the NCSA httpd server. Netsite software allows organizations to "serve" or post information, including formatted text, graphical images, audio and video, and to conduct transactions on the Internet and other global networks based on the Internet Protocol (IP).

The Netsite product line includes two products available for UNIX-based platforms:

  • Netsite Communications Server
  • Netsite Commerce Server

The Netsite Communications Server is designed for applications that don't require the advanced security features contained in the Netsite Commerce Server, such as the delivery of online marketing materials and customer support information to customers, and communications to employees.

The Netsite Commerce Server makes electronic commerce a reality allowing transactions and the exchange of sensitive data on the Internet. The Netsite Commerce Server provides bulletproof security based on public-key cryptographic technology from RSA Data Security, Inc.

Bulletproof security includes data encryption to ensure the privacy of client/server communications, and server authentication, which uses certification and a digital signature to verify the legitimacy of the server.

What can a Netsite server software do for my business?

Netsite servers let you reach the more than 3 million users of HTTP-compatible network navigators (including Netscape and NCSA Mosaic) who access the Internet; or, simply communicate with the people in your branch office.

With the Netsite Communication Server you can set up an interactive, media-rich environment for applications such as marketing, corporate communications, and distance learning.

Using the Netsite Commerce Server, you can close the deal online, accepting both orders and payments over the Internet. The Netsite Commerce Server can also handle online publishing services and sensitive communications of all kinds.

How is Netsite server software different from the NCSA httpd server software?

Here are some of the features that makes Netsite server software outstanding:
  • Increased performance, including lightning fast response times and superior handling of peak loads
  • More efficient use of your hardware and communications infrastructure, reducing the total cost of operation
  • Easier installation, configuration and maintenance using intuitive online forms
  • Bulletproof security, making it possible to conduct commerce and secure communications using the Netsite Commerce Server
  • Scalability, to accommodate varying traffic loads
  • Extensibility, to incorporate additional features on a modular basis

Netsite servers are also built and backed by the development team that conceived and wrote the NCSA httpd server prototype and are supported by a professional software support organization.

Are Netsite servers compatible with NCSA httpd and NCSA Mosaic? Do they support industry standard protocols and document formats?

Netsite servers are backward compatible with all HTTP-based clients and servers, including NCSA Mosaic and NCSA httpd respectively. They utilize industry standard protocols HTTP and TCP/IP and the HTML document format. You can serve any hypermedia content you may have already developed.

Does Netsite software provide security for online transactions or sensitive communications?

The Netsite Commerce Server incorporates bulletproof security based upon public-key cryptographic technology from RSA Data Security, the leader in computer data security.

Bulletproof security includes:

  • Encryption to ensure the privacy of client/server communications
  • Server authentication to verify the identity of the server

The Netsite Communications Server does not have any security features.

What are the hardware and software requirements for using Netsite software?

Netsite servers can operate on the following UNIX platforms:

    • Sun (Solaris 2.3 and 2.4, SunOS 4.1.3)
    • Silicon Graphics (IRIX 5.2)
    • Digital Equipment Corporation (OSF/1 2.0)
    • IBM (AIX 3.2.5)
    • Hewlett-Packard (HP-UX 9.0.3)
    • 386/486 Intel Pentium PCs (BSDI 1.1)
