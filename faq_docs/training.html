Training

Mosaic Communications is proud to announce the following training program:

  • Netsite Server Administration
Course Description:
This two day course is intended for UNIX system administrators who are implementing an Internet HTTP presence with Mosaic Communications' Netsite software. Topic areas contained in the course are:

  • Overview of Mosaic Communications and the Internet
  • Netsite structure and demonstration of key features
  • Installation of Netsite software
  • User support for your new Netsite location
  • Server manager functions and resource center security optimization
  • CGI authoring
  • Secure transactions using Netsite and Netscape
  • Activating your server for secure sessions
  • Secure module integration
  • HTML authoring
  • Customized home page design
  • Customer Solutions Center services

    • Sessions will be interactive and emphasize a hands-on learning approach. Our instructors are fluent in all forms of UNIX and use of various platforms including Sun, DEC, SGI, IBM, and HP platforms.
