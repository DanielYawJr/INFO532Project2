Guided Tour Tutorials - FTP servers FTP Servers
Netscape makes browsing FTP servers very easy. With it you may navigate directories, view files, and download useful programs using a simple point and click interface.

When possible Netscape shows the type, size, date and a short description of each file in a directory. Clicking on a directory link moves you into that directory. Clicking on a text file displays the file, and clicking on a binary file or program will automatically download the file or program and put it on your local computer in a directory of your choosing.

An almost infinite number of interesting programs, utilities, helpful advice and even games can be found on Internet FTP sites. Below is a sample listing of the popular internet FTP archive wuarchive.wustl.edu. The wuarchive FTP site is the most popular and most visited FTP site in the world. It contains an extensive archive of DOS, Windows, Macintosh and UNIX programs and utilities-more than 3,000 Megabytes of data in all. And all the programs and utilities can be downloaded free of charge. Note that you may have a hard time getting connected to wuarchive at times since it is so popular and has a preset limit on the number of people that may visit it at any one time.
